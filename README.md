# MajorProj

MajorProj is an AI-driven waste management system based on circular economy principles.
It uses Machine Learning and Large Language Models (LLMs) to optimize waste pickup,
routing, and forecasting.

## Tech Stack
- Python (FastAPI)
- Machine Learning (XGBoost, LightGBM)
- LLMs (OpenAI API)
- Frontend & Backend integration (future scope)

## SDG Alignment
- SDG 12: Responsible Consumption and Production
- SDG 13: Climate Action
